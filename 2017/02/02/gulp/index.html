Gulp

gulp是一个前端自动化的构建工具。
其优点是：
  1、避免缓存误会：js和css属于静态文件，很多时候浏览器存在缓存机制。利用构建工具，给每一个静态文件添加一个版本号，浏览器就会认为是新的文件，就不存在缓存机制。
  2、性能优化：文件合并，减少http请求；文件压缩，减少文件体积，加快下载速度；
  3、效率提升：自动添加CSS3的vendor前缀；代码分析检查改正；在使用MVC和MVVM的框架后，可在提交之前，使用gulp自动跑一遍单元测试；

安装步骤

1、利用nodejs安装

2、在项目上安装项目依赖（devDependencies）

3、在项目根目录中创建gulpfile.js文件

4、运行

Gulp的API

主要为gulp.task(),gulp.src(),gulp.dest(),gulp.watch()四个

1、gulp.src()
gulp.src()用来读取你需要操作的文件

globs参数是文件匹配模式(类似正则表达式)，用来匹配文件路径(包括文件名)，当然这里也可以直接指定某个具体的文件路径。当有多个匹配模式时，该参数可以为一个数组。
options为可选参数。通常情况下我们不需要用到。

3、gulp.task()
gulp.task()方法用来定义任务，其语法为：

name 为任务名
deps 是当前定义的任务需要依赖的其他任务，为一个数组。当前定义的任务会在所有依赖的任务执行完毕后才开始执行。如果没有依赖，则可省略这个参数
fn 为任务函数，我们把任务要执行的代码都写在里面。该参数也是可选的。

常用gulp插件

1、js文件压缩gulp-uglify
安装：npm install --save-dev gulp-uglify

2、css文件压缩gulp-minify-css
安装：npm install --save-dev gulp-minify-css

3、html文件压缩gulp-minify-html
安装：npm install --save-dev gulp-minify-css

4、js代码检查gulp-jshint
安装：npm install --save-dev gulp-jshint

5、文件合并gulp-concat
安装：npm install --save-dev gulp-concat

6、less和sass的编译
less使用gulp-less
安装：npm install --save-dev gulp-less
sass使用gulp-sass
安装：npm install --save-dev gulp-sass

7、图片压缩gulp-imagemin
安装：npm install --save-dev gulp-imagemin

8、自动加插件gulp-load-plugins
安装：npm install --save-dev gulp-load-plugins
